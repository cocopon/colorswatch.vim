colorswatch.vim
===============


Overview
--------
colorswatch.vim generates a beautiful color swatch of the current buffer.

![Screenshot](http://cocopon.me/app/colorswatch.vim/img/screenshot.png)


How to Use
----------
To generate a color swatch, just type the following commnand:

```vim
:ColorSwatchGenerate
```
